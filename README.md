# 🎙️ Fake Voice Detection using Deep Learning

A deep learning project for detecting **spoofed or AI-generated speech** using audio features and ensemble learning models.
The system is trained and evaluated on the **ASVspoof2019 Logical Access (LA)** dataset.

---

## 📌 Overview

With the rapid development of **AI voice cloning and speech synthesis**, detecting fake or manipulated speech has become an important research problem.

This project focuses on building a **robust spoofing detection system** using:

* Audio feature extraction (MFCC, CQT)
* Deep learning architectures
* Ensemble stacking models
* Evaluation using standard spoofing metrics

The goal is to classify speech as:

* **Real (bonafide)**
* **Fake / Spoofed**

---

## 🧠 Models Used

The project implements multiple deep learning models:

* WaveNet
* ResNet
* LSTM
* Transformer

These models are combined using a **stacking ensemble approach** where:

* Base models generate prediction probabilities
* A **meta-classifier (Logistic Regression)** learns to combine them

---

## 📂 Dataset

Dataset used:

**ASVspoof2019 Logical Access (LA)**

It contains:

* Real human speech
* Spoofed speech generated by various synthesis and voice conversion attacks

Dataset structure example:

```
ASVspoof2019_LA/
│
├── train/
├── dev/
├── eval/
└── protocols/
```

Audio format: `.flac`

---

## ⚙️ Feature Extraction

Two main audio features are used:

### MFCC (Mel-Frequency Cepstral Coefficients)

Captures spectral characteristics of speech.

### CQT (Constant-Q Transform)

Better frequency resolution for speech analysis.

Libraries used:

* librosa
* nnAudio
* numpy

---

## 🧪 Data Augmentation

To improve model generalization, the following augmentation methods are applied:

* Mixup
* CutMix
* ShuffleMix
* SpecAugment

---

## 📊 Evaluation Metrics

Model performance is evaluated using:

* **Accuracy**
* **F1-score**
* **AUC (Area Under ROC Curve)**
* **EER (Equal Error Rate)**

EER is a standard metric in **spoofing detection research**.

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/fake-voice-detection.git
cd fake-voice-detection
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## 🏋️ Training

Train individual models:

```bash
python train_resnet.py
python train_transformer.py
python train_lstm.py
python train_wavenet.py
```

Train stacking ensemble:

```bash
python train_stacking.py
```

---

## 📈 Evaluation

Evaluate model performance:

```bash

```
